# ai

This is a javascript npm package for using llm inside your code. It produces both `cjs` and `esm` module builds.

## Installation

- Using `npm`

```bash
npm i @pr4j3sh/ai
```

## Usage

```js
const { ai } = require("@pr4j3sh/ai");

console.log(ai("write a poem on javascript"));
```
